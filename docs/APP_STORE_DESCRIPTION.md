# Shillings — App Store Description

## Short Description
Professional double-entry accounting for your business, available offline.

## Full Description
Shillings is a professional double-entry bookkeeping app designed for small businesses, freelancers, and accountants. Built with precision-based fraction arithmetic (no floating point errors), Shillings ensures your books are always accurate.

### Key Features

**Double-Entry Accounting**
- Full chart of accounts with hierarchical account types
- Multi-split transactions with real-time balance validation
- GnuCash-style fraction-based arithmetic for penny-perfect accuracy

**Offline-First**
- Works completely offline with automatic background sync
- IndexedDB local storage with conflict resolution
- Never lose a transaction — queue changes when disconnected

**Business Management**
- Contacts: customers, vendors, and employees
- Invoicing with line items and automatic totals
- Document management with receipt capture

**Financial Reports**
- Balance Sheet, Income Statement, Cash Flow, Trial Balance
- Beautiful interactive charts (pie, bar, line)
- Account register with transaction history

**Budgeting & Planning**
- Budget tracking with variance analysis
- Scheduled recurring transactions
- Low balance alerts and reminders

**Multi-Company Support**
- Switch between multiple businesses instantly
- Per-company chart of accounts and reports
- Role-based access control

**Security**
- Biometric authentication (Face ID / Touch ID)
- Encrypted token storage
- Automatic session management

### About
Built by Advent Digital. Shillings is the mobile companion to the Shillings web accounting platform, sharing the same backend API and offline sync engine.
