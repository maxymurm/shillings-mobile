## Full Description (4000 chars max)
Shillings is a professional double-entry bookkeeping app built for small businesses, freelancers, and accountants who demand accuracy. Using fraction-based arithmetic instead of floating-point, Shillings guarantees penny-perfect calculations every time.

★ DOUBLE-ENTRY ACCOUNTING
• Full chart of accounts with hierarchical types (Assets, Liabilities, Equity, Income, Expenses)
• Multi-split transactions with real-time debit/credit balance validation
• GnuCash-style fraction arithmetic — no rounding errors, ever

★ WORKS OFFLINE
• Full offline functionality — create transactions without internet
• Automatic background sync when connection returns
• Smart conflict resolution for concurrent edits
• IndexedDB local storage keeps your data safe

★ COMPLETE BUSINESS MANAGEMENT
• Contact management: customers, vendors, employees
• Professional invoicing with line items
• Receipt capture with camera or photo gallery
• Document management and file attachments

★ POWERFUL REPORTING
• Balance Sheet with interactive pie chart
• Income Statement with bar chart comparison
• Cash Flow analysis with trend lines
• Trial Balance with drill-down
• Account register with full transaction history

★ BUDGETING & PLANNING
• Set budgets per account and track variance
• Scheduled recurring transactions
• Due date reminders via push notifications
• Low balance alerts

★ MULTI-COMPANY READY
• Switch between multiple businesses instantly
• Separate chart of accounts per company
• Role-based access control

★ SECURITY FIRST
• Biometric login (fingerprint / face recognition)
• Encrypted API communication
• No ads, no tracking

Built by Advent Digital. Shillings mobile is the companion app to the Shillings web accounting platform.
